# 1일차 응용 실습 — 마지막 1시간 도전 프로젝트 4

> 1일차에 배운 **내장 LED 매트릭스 · 버튼 · 마이크 · 가속도 센서**만으로 만드는 ‘작은 작품’ 4가지입니다.
> 외부 부품이 전혀 없습니다. 각 실습은 **약 15분**, 단계별로 코드를 조금씩 키워 가며 따라 하면 됩니다.

| # | 실습 | 사용 요소 | 난이도 |
|---|------|-----------|--------|
| 1 | 감정 표현 배지 | LED + 버튼 + 가속도 | ★ |
| 2 | 박수로 켜는 스마트 전등 | LED + 마이크 | ★★ |
| 3 | 가위바위보 게임기 | LED + 버튼 (+ 가속도) | ★★ |
| 4 | LED 모래시계 타이머 | LED + 버튼 | ★★ |

## 공통 준비 (꼭 먼저 읽기)

- **LED 좌표** — 5×5 매트릭스, `index = 행(row)×5 + 열(col)` (0=좌상 → 24=우하)
- **버튼** — A = `Pin(36)`, B = `Pin(39)`. 누르면 `0`, 안 누르면 `1`
- **밝기** — RGB 값은 `10~20` 정도로 낮게 (너무 밝으면 눈부심·전류↑)
- **가속도(실습 1·3에서 사용)** — `mpu6050.py`가 보드에 업로드되어 있어야 하고, `i2c.scan()` 결과에 **105(0x69)** 가 보여야 합니다.

> **그림(패턴) 그리는 법** — 아래 실습들은 5줄짜리 문자열로 그림을 표현합니다. `'1'`은 켜는 칸, `'0'`은 끄는 칸입니다.

```python
# 모든 실습에서 쓰는 '그림 그리기' 도우미
from neopixel import NeoPixel
from machine import Pin
np = NeoPixel(Pin(13, Pin.OUT), 25)

def draw(grid, color=(0, 12, 0)):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()
```

# 실습 1. 감정 표현 배지 😊

> **목표** — 버튼을 누를 때마다 표정이 바뀌는 배지. 흔들면 ‘어지러운’ 표정이 됩니다.

### 단계 1 — 표정 하나 그려보기

먼저 그림 도우미로 웃는 얼굴을 띄워 봅니다.

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)

def draw(grid, color=(0, 12, 0)):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

SMILE = ['01010',
         '01010',
         '00000',
         '10001',
         '01110']
draw(SMILE)
```

✅ **실행 결과** — 5×5 LED에 웃는 얼굴이 나타납니다.

### 단계 2 — 여러 표정 만들고 버튼으로 넘기기

표정 목록을 만들고, **A 버튼**을 누를 때마다 다음 표정으로 바꿉니다.

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)
btn_a = Pin(36, Pin.IN)

def draw(grid, color=(0, 12, 0)):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

SMILE = ['01010','01010','00000','10001','01110']
SAD   = ['01010','01010','00000','01110','10001']
HEART = ['01010','11111','11111','01110','00100']

faces  = [SMILE, SAD, HEART]
colors = [(0,12,0), (0,0,14), (14,0,4)]
idx = 0
draw(faces[idx], colors[idx])

while True:
    if btn_a.value() == 0:           # A 누름
        idx = (idx + 1) % len(faces)  # 다음 표정
        draw(faces[idx], colors[idx])
        sleep(0.3)                    # 한 번만 넘어가게
```

✅ **실행 결과** — A를 누를 때마다 웃음 → 슬픔 → 하트 순으로 표정이 바뀝니다.

### 단계 3 — 흔들면 ‘어지러움’ 표정 (가속도)

> ⚠️ `mpu6050.py`가 업로드되어 있어야 합니다.

```python
from machine import I2C, Pin
from neopixel import NeoPixel
import mpu6050, math
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)
btn_a = Pin(36, Pin.IN)
i2c = I2C(0, scl=Pin(21), sda=Pin(22))
imu = mpu6050.accel(i2c, 0x69)

def draw(grid, color=(0, 12, 0)):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

SMILE = ['01010','01010','00000','10001','01110']
SAD   = ['01010','01010','00000','01110','10001']
HEART = ['01010','11111','11111','01110','00100']
DIZZY = ['10101','01010','00100','01010','10101']

faces  = [SMILE, SAD, HEART]
colors = [(0,12,0), (0,0,14), (14,0,4)]
idx = 0
draw(faces[idx], colors[idx])

while True:
    a = imu.get_values()
    shake = math.sqrt(a['AcX']**2 + a['AcY']**2 + a['AcZ']**2)
    if shake > 28000:                 # 흔들림 감지
        draw(DIZZY, (14, 10, 0))      # 어지러운 표정
        sleep(0.6)
        draw(faces[idx], colors[idx])
    if btn_a.value() == 0:
        idx = (idx + 1) % len(faces)
        draw(faces[idx], colors[idx])
        sleep(0.3)
    sleep(0.02)
```

✅ **실행 결과** — 버튼으로 표정을 바꾸고, 보드를 흔들면 잠깐 ‘어지러운’ 표정이 떴다가 돌아옵니다.

🎯 **도전** — 나만의 표정(화남·놀람·윙크)을 추가해 보세요. 표정마다 색을 다르게!

# 실습 2. 박수로 켜는 스마트 전등 👏

> **목표** — 박수를 치면 불이 켜지고, 다시 치면 꺼지는 ‘박수 전등’.

### 단계 1 — 소리 크기 읽어보기

마이크로 소리 크기를 측정해 출력합니다. (박수의 기준값을 정하기 위해)

```python
from machine import Pin, ADC
from time import sleep
mic = ADC(Pin(35))
mic.atten(ADC.ATTN_11DB)
while True:
    s = [mic.read() for _ in range(100)]
    level = max(s) - min(s)
    print('소리 크기:', level)
    sleep(0.1)
```

✅ **확인** — 조용할 때와 **박수칠 때**의 숫자를 비교하세요. 박수 때 값(예: 500~2000)을 기억해 **기준값(THRESHOLD)** 으로 씁니다.

### 단계 2 — 박수 감지하기

기준값을 넘으면 “박수!”를 출력합니다. (한 번의 박수가 여러 번 잡히지 않게 잠깐 쉼)

```python
from machine import Pin, ADC
from time import sleep
mic = ADC(Pin(35)); mic.atten(ADC.ATTN_11DB)
THRESHOLD = 600          # 단계 1에서 확인한 값으로 조절

while True:
    s = [mic.read() for _ in range(60)]
    level = max(s) - min(s)
    if level > THRESHOLD:
        print('박수!')
        sleep(0.3)        # 중복 감지 방지
```

### 단계 3 — 박수로 LED 켜고 끄기 (토글)

박수를 칠 때마다 전체 LED가 켜졌다 꺼집니다.

```python
from machine import Pin, ADC
from neopixel import NeoPixel
from time import sleep
mic = ADC(Pin(35)); mic.atten(ADC.ATTN_11DB)
np = NeoPixel(Pin(13, Pin.OUT), 25)
THRESHOLD = 600
on = False

def fill(color):
    for i in range(25): np[i] = color
    np.write()

fill((0, 0, 0))
while True:
    s = [mic.read() for _ in range(60)]
    level = max(s) - min(s)
    if level > THRESHOLD:
        on = not on                       # 상태 뒤집기
        fill((12, 12, 8) if on else (0, 0, 0))
        print('전등', 'ON' if on else 'OFF')
        sleep(0.3)
```

✅ **실행 결과** — 박수 한 번 → 불 ON, 다시 박수 → 불 OFF.

🎯 **도전** — **2번 연속 박수**일 때만 토글되게 만들기(오작동 ↓). 켤 때 색을 무지개로 바꾸기. 박수 횟수를 LED 개수로 표시하기.

# 실습 3. 가위바위보 게임기 ✊✌🖐

> **목표** — 버튼을 누르면 가위·바위·보 중 하나가 무작위로 나오는 게임기.

### 단계 1 — 세 가지 모양 그리기

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)

def draw(grid, color=(0, 0, 14)):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

ROCK     = ['00000','01110','01110','01110','00000']   # 바위(주먹)
SCISSORS = ['10001','10001','01010','00100','00100']   # 가위
PAPER    = ['11111','10001','10001','10001','11111']   # 보

draw(ROCK)
```

✅ **확인** — `draw(ROCK)`, `draw(SCISSORS)`, `draw(PAPER)`를 하나씩 실행해 모양을 확인하세요.

### 단계 2 — 버튼 누르면 무작위로 내기

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
import random
np = NeoPixel(Pin(13, Pin.OUT), 25)
btn_a = Pin(36, Pin.IN)

def draw(grid, color=(0, 0, 14)):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

ROCK     = ['00000','01110','01110','01110','00000']
SCISSORS = ['10001','10001','01010','00100','00100']
PAPER    = ['11111','10001','10001','10001','11111']
hands = [ROCK, SCISSORS, PAPER]

while True:
    if btn_a.value() == 0:
        choice = random.choice(hands)
        draw(choice, (0, 10, 12))
        sleep(0.4)
```

### 단계 3 — ‘두근두근’ 애니메이션 추가

결과가 나오기 전에 모양이 빠르게 바뀌어 긴장감을 줍니다.

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
import random
np = NeoPixel(Pin(13, Pin.OUT), 25)
btn_a = Pin(36, Pin.IN)

def draw(grid, color=(0, 0, 14)):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

ROCK     = ['00000','01110','01110','01110','00000']
SCISSORS = ['10001','10001','01010','00100','00100']
PAPER    = ['11111','10001','10001','10001','11111']
hands = [ROCK, SCISSORS, PAPER]

while True:
    if btn_a.value() == 0:
        for _ in range(8):                      # 빠르게 섞기
            draw(random.choice(hands), (12, 8, 0))
            sleep(0.08)
        draw(random.choice(hands), (0, 10, 12)) # 최종 결과
        sleep(0.6)
```

✅ **실행 결과** — 버튼을 누르면 모양이 다다닥 바뀌다가 하나로 ‘짠!’ 멈춥니다.

🎯 **도전** — **흔들어서** 내기(`mpu6050`의 흔들림 감지 사용). 친구와 2대로 동시에 내서 승부 가리기. 결과 모양을 더 멋지게 디자인하기.

# 실습 4. LED 모래시계 타이머 ⏳

> **목표** — 버튼을 누르면 25개 LED가 ‘모래’처럼 하나씩 사라지는 카운트다운 타이머.

### 단계 1 — LED를 하나씩 끄기 (모래 떨어지기)

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)

for i in range(25): np[i] = (12, 8, 0)   # 전체 켜기(모래 가득)
np.write()
sleep(0.5)

for i in range(24, -1, -1):              # 위에서부터 하나씩 비우기
    np[i] = (0, 0, 0)
    np.write()
    sleep(0.2)
```

✅ **실행 결과** — 가득 찬 LED가 0.2초마다 하나씩 사라집니다.

### 단계 2 — 원하는 시간(초)에 맞추기

전체 시간을 25등분해서 ‘한 칸 = 전체시간/25초’로 만듭니다.

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)

SECONDS = 10                  # 총 10초 타이머
step = SECONDS / 25           # 한 칸당 시간

for i in range(25): np[i] = (12, 8, 0)
np.write()
for i in range(24, -1, -1):
    np[i] = (0, 0, 0)
    np.write()
    sleep(step)
```

### 단계 3 — 버튼으로 시작 + 끝나면 빨강 깜빡이

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)
btn_a = Pin(36, Pin.IN)
SECONDS = 10
step = SECONDS / 25

def fill(color):
    for i in range(25): np[i] = color
    np.write()

fill((0, 0, 0))
print('A를 누르면 타이머 시작')

while True:
    if btn_a.value() == 0:               # 시작
        fill((12, 8, 0))                  # 모래 가득
        sleep(0.3)
        for i in range(24, -1, -1):
            np[i] = (0, 0, 0)
            np.write()
            sleep(step)
        for _ in range(6):                # 끝! 빨강 깜빡
            fill((14, 0, 0)); sleep(0.25)
            fill((0, 0, 0));  sleep(0.25)
    sleep(0.05)
```

✅ **실행 결과** — A를 누르면 모래가 줄어들고, 다 떨어지면 빨갛게 깜빡여 알려줍니다.

🎯 **도전** — **B 버튼**으로 시간(5/10/30초) 바꾸기. 마지막 5칸은 노랑·빨강으로 경고색 표시. 끝날 때 마이크 대신 ‘삐’ 소리 패턴(LED)로 알리기.
