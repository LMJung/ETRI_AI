## 실습 1. 감정 표현 배지 — 도전 과제

**과제** — 나만의 표정(화남·놀람·윙크)을 추가하고, 표정마다 색을 다르게.

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)
btn_a = Pin(36, Pin.IN)

def draw(grid, color):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

# --- 표정 추가 ---
SMILE    = ['01010','01010','00000','10001','01110']
SAD      = ['01010','01010','00000','01110','10001']
HEART    = ['01010','11111','11111','01110','00100']
ANGRY    = ['10001','01010','00000','01110','10001']   # 화남(눈썹 ↘↙ + 찡그림)
SURPRISE = ['01010','00000','01110','01010','01110']   # 놀람(입 O)
WINK     = ['00011','00010','00000','10001','01110']   # 윙크(한쪽 눈 감음)

# 표정과 '각자 다른 색'을 짝지어 둠
faces  = [SMILE,      SAD,      HEART,    ANGRY,    SURPRISE,  WINK]
colors = [(0,12,0), (0,0,14), (14,0,4), (14,2,0), (12,8,0),  (0,12,10)]

idx = 0
draw(faces[idx], colors[idx])
while True:
    if btn_a.value() == 0:                 # A = 다음 표정
        idx = (idx + 1) % len(faces)
        draw(faces[idx], colors[idx])
        sleep(0.3)
```

> **포인트** — 표정과 색을 같은 순서의 두 리스트(`faces`, `colors`)로 묶어 `idx` 하나로 함께 바꿉니다. 5×5 픽셀이 작으니 그림은 자유롭게 다듬어 보세요.

## 실습 2. 박수로 켜는 스마트 전등 — 도전 과제

### (1) 2번 연속 박수일 때만 토글 + 켤 때 무지개색

```python
from machine import Pin, ADC
from neopixel import NeoPixel
from time import sleep, ticks_ms, ticks_diff
mic = ADC(Pin(35)); mic.atten(ADC.ATTN_11DB)
np = NeoPixel(Pin(13, Pin.OUT), 25)
THRESHOLD = 600          # 단계 1에서 확인한 값으로 조절
on = False
last_clap = 0

def wheel(pos):                       # 0~255 → 무지개 색
    if pos < 85:    r, g, b = pos*3, 255 - pos*3, 0
    elif pos < 170: pos -= 85;  r, g, b = 255 - pos*3, 0, pos*3
    else:           pos -= 170; r, g, b = 0, pos*3, 255 - pos*3
    return (r // 14, g // 14, b // 14)   # 밝기 낮추기

def rainbow():
    for i in range(25): np[i] = wheel((i * 10) % 255)
    np.write()

def off():
    for i in range(25): np[i] = (0, 0, 0)
    np.write()

off()
while True:
    s = [mic.read() for _ in range(60)]
    if max(s) - min(s) > THRESHOLD:           # 박수 감지
        now = ticks_ms()
        if ticks_diff(now, last_clap) < 600:  # 0.6초 안에 두 번째 박수
            on = not on
            rainbow() if on else off()
            last_clap = 0                     # 3연속 방지로 리셋
        else:
            last_clap = now                   # 첫 박수 시각 기록
        sleep(0.25)                           # 같은 박수 중복 방지
```

> **포인트** — 박수 시각을 `ticks_ms()`로 기록하고, 다음 박수가 **0.6초 안에** 들어오면 ‘두 번 박수’로 인정해 토글합니다. 혼자 친 한 번의 박수로는 안 켜져 오작동이 줄어듭니다.

### (2) 박수 횟수를 LED 개수로 표시 (보너스)

```python
from machine import Pin, ADC
from neopixel import NeoPixel
from time import sleep
mic = ADC(Pin(35)); mic.atten(ADC.ATTN_11DB)
np = NeoPixel(Pin(13, Pin.OUT), 25)
THRESHOLD = 600
count = 0

while True:
    s = [mic.read() for _ in range(60)]
    if max(s) - min(s) > THRESHOLD:
        count = (count + 1) % 26          # 0~25 순환
        for i in range(25):
            np[i] = (0, 10, 0) if i < count else (0, 0, 0)
        np.write()
        print('박수 횟수:', count)
        sleep(0.25)
```

> 박수를 칠 때마다 켜진 LED가 한 칸씩 늘어납니다(25개를 넘으면 0으로 초기화).

## 실습 3. 가위바위보 게임기 — 도전 과제

### (1) 흔들어서 내기 (가속도)

> `mpu6050.py` 업로드 필요. 버튼 대신 보드를 흔들면 가위바위보가 나옵니다.

```python
from machine import I2C, Pin
from neopixel import NeoPixel
import mpu6050, math, random
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)
i2c = I2C(0, scl=Pin(21), sda=Pin(22))
imu = mpu6050.accel(i2c, 0x69)

def draw(grid, color):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

ROCK     = ['00000','01110','01110','01110','00000']
SCISSORS = ['10001','10001','01010','00100','00100']
PAPER    = ['11111','10001','10001','10001','11111']
hands = [ROCK, SCISSORS, PAPER]

while True:
    a = imu.get_values()
    if math.sqrt(a['AcX']**2 + a['AcY']**2 + a['AcZ']**2) > 28000:   # 흔들림
        for _ in range(8):                       # 두근두근
            draw(random.choice(hands), (12, 8, 0)); sleep(0.08)
        draw(random.choice(hands), (0, 10, 12))  # 결과
        sleep(0.8)
    sleep(0.02)
```

### (2) 2인 승부 — 판정 기능 (한 보드로)

> **A = 1번 플레이어**, **B = 2번 플레이어**가 각각 무작위로 내고, 보드가 승자를 알려줍니다.

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
import random
np = NeoPixel(Pin(13, Pin.OUT), 25)
btn_a = Pin(36, Pin.IN); btn_b = Pin(39, Pin.IN)

def draw(grid, color):
    for r in range(5):
        for c in range(5):
            np[r*5 + c] = color if grid[r][c] == '1' else (0, 0, 0)
    np.write()

ROCK     = ['00000','01110','01110','01110','00000']
SCISSORS = ['10001','10001','01010','00100','00100']
PAPER    = ['11111','10001','10001','10001','11111']
hands = [ROCK, SCISSORS, PAPER]          # 0=바위, 1=가위, 2=보

def judge(a, b):                          # 0=무승부, 1=a승, 2=b승
    if a == b: return 0
    return 1 if (a + 1) % 3 == b else 2   # 바위>가위>보>바위

def flash_side(left):                     # 승자 쪽 절반 깜빡
    for _ in range(4):
        for i in range(25):
            col = (0, 12, 0)
            np[i] = col if ((i % 5) < 2) == left else (0, 0, 0)
        np.write(); sleep(0.2)
        for i in range(25): np[i] = (0, 0, 0)
        np.write(); sleep(0.2)

p1 = p2 = None
while True:
    if btn_a.value() == 0:
        p1 = random.randint(0, 2); draw(hands[p1], (0, 0, 12)); sleep(0.3)
    if btn_b.value() == 0:
        p2 = random.randint(0, 2); draw(hands[p2], (12, 0, 0)); sleep(0.3)
    if p1 is not None and p2 is not None:
        w = judge(p1, p2)
        if   w == 0: draw(['00000','00100','01110','00100','00000'], (12,12,0))  # 무승부
        elif w == 1: flash_side(True)    # 1P(왼쪽) 승
        else:        flash_side(False)   # 2P(오른쪽) 승
        sleep(0.6); p1 = p2 = None       # 다음 판
```

> **포인트** — `judge()`가 핵심입니다. 바위(0)>가위(1)>보(2)>바위(0) 규칙을 `(a+1) % 3 == b`로 한 줄에 판정합니다. 결과 모양·색은 자유롭게 더 멋지게 디자인해 보세요.

## 실습 4. LED 모래시계 타이머 — 도전 과제

**과제** — B 버튼으로 시간(5/10/30초) 바꾸기 + 마지막 5칸 경고색 + 끝나면 알림 패턴.

```python
from neopixel import NeoPixel
from machine import Pin
from time import sleep
np = NeoPixel(Pin(13, Pin.OUT), 25)
btn_a = Pin(36, Pin.IN); btn_b = Pin(39, Pin.IN)

DURATIONS = [5, 10, 30]      # B로 고르는 시간
di = 1                       # 시작은 10초

def fill(color):
    for i in range(25): np[i] = color
    np.write()

def show_seconds(sec):       # 선택한 시간을 잠깐 보여주기(켜진 칸 수 = 초/2 정도)
    n = min(sec // 2, 25)
    for i in range(25): np[i] = (0, 0, 12) if i < n else (0, 0, 0)
    np.write(); sleep(0.6)

fill((0, 0, 0))
print('B=시간 변경, A=시작')

while True:
    if btn_b.value() == 0:                       # 시간 변경
        di = (di + 1) % len(DURATIONS)
        print('설정 시간:', DURATIONS[di], '초')
        show_seconds(DURATIONS[di])
        fill((0, 0, 0)); sleep(0.3)

    if btn_a.value() == 0:                        # 시작
        seconds = DURATIONS[di]
        step = seconds / 25
        for left in range(25, 0, -1):             # 남은 칸 수
            if   left <= 5:  col = (14, 0, 0)     # 경고: 빨강
            elif left <= 10: col = (14, 10, 0)    # 주의: 노랑
            else:            col = (12, 8, 0)     # 평소: 주황
            for i in range(25):
                np[i] = col if i < left else (0, 0, 0)
            np.write()
            sleep(step)
        for _ in range(6):                        # 끝! 알림 깜빡임
            fill((14, 0, 0)); sleep(0.25)
            fill((0, 0, 0));  sleep(0.25)
    sleep(0.05)
```

> **포인트**
> - **B 버튼**: `DURATIONS` 목록을 순환하며 시간을 바꾸고, 선택한 시간을 LED로 잠깐 표시.
> - **경고색**: 남은 칸이 10개 이하면 노랑, 5개 이하면 빨강.
> - **알림**: 다 떨어지면 빨강 전체 점멸. (스피커가 있다면 점멸 자리에 소리를 넣어 ‘삐-’ 효과 가능)

## 마무리

각 도전 과제의 핵심 아이디어:

- **실습 1** — 표정과 색을 두 리스트로 짝지어 함께 전환
- **실습 2** — `ticks_ms()`로 ‘두 번 박수’를 시간으로 구분 / 무지개 = `wheel()` 함수
- **실습 3** — `judge()` 한 줄로 가위바위보 승부 판정 / 가속도 흔들림으로 입력
- **실습 4** — 남은 칸 수에 따라 색을 바꾸고, 끝에 점멸로 알림

> 모든 솔루션은 ‘정답 예시’일 뿐입니다. 학생들이 색·패턴·임계값·시간 등을 바꿔 **자기만의 버전**으로 발전시키도록 지도해 주세요.
